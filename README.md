# 🎮 坦克大战 (Tank Battle)

经典 FC 红白机坦克大战的 HTML5 Canvas 复刻版。

## 🕹️ 游戏玩法

- 控制你的黄色坦克，消灭所有敌方坦克
- **保护底部中央的基地（星形标志）**，基地被摧毁则游戏结束
- 每关敌人数量随关卡递增，敌人类型也会变化

## 🎛️ 操作说明

| 按键 | 功能 |
|------|------|
| `↑ ↓ ← →` 或 `W A S D` | 移动坦克 |
| `空格` | 发射子弹 |
| `回车` | 开始 / 重新开始游戏 |
| `P` | 暂停 / 继续 |

## 🧱 地图元素

| 元素 | 说明 |
|------|------|
| 🧱 砖墙（棕色） | 可被子弹摧毁 |
| ⚙️ 钢墙（灰色） | 不可摧毁 |
| 🌊 水（蓝色） | 不可通过 |
| 🌳 树丛（绿色） | 可通过，遮挡视野 |
| ⭐ 基地（红色星） | 必须保护！被摧毁则游戏结束 |

## 👾 敌人类型

| 类型 | 颜色 | 特点 | 分数 |
|------|------|------|------|
| 普通坦克 | 暗红色 | 标准速度，1点生命 | 100 |
| 快速坦克 | 橙色 | 移动更快 | 200 |
| 重型坦克 | 亮红色 | 移动慢，2点生命 | 300 |

## 🚀 技术栈

- HTML5 Canvas
- 原生 JavaScript（无框架）
- Web Audio API（简易音效）
- 响应式触屏支持

## 📂 运行方式

直接在浏览器中打开 `index.html` 即可游玩。

```bash
# 或使用任意本地服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```
